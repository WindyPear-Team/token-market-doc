# POST `/v1/midjourney/generations/imagine`

Midjourney Imagine 显式入口。行为与 [POST /v1/midjourney/generations](/api/ai/routes/v1-midjourney-generations) 相同，但路径更明确。

## 请求

| 项 | 内容 |
| --- | --- |
| 方法 | `POST` |
| 路径 | `/v1/midjourney/generations/imagine` |
| 鉴权 | API Key 或登录 JWT |
| Content-Type | `application/json` |

## 参数与返回值

同 [POST /v1/midjourney/generations](/api/ai/routes/v1-midjourney-generations)。
