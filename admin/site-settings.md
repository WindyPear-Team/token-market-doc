# 站点与界面设置

平台的品牌、文案、导航和界面开关都存放在数据库系统设置里，在后台「系统设置」中修改。本页汇总常用的展示类设置项。运行参数（端口、数据库、JWT 等）见 [环境变量](/admin/configuration)。

## 品牌与文案

| 设置键 | 默认 | 说明 |
| --- | --- | --- |
| `站点名称` | `flai` | 站点名称 |
| `Base URL` | 空 | 站点对外基础地址，用于拼接支付回调等，**务必设为对外 HTTPS 域名** |
| `图标 URL` | 空 | 站点图标地址 |
| `页脚文本` | 空 | 页脚文案 |
| `关于页面 HTML` | 空 | 「关于」页 HTML 内容 |
| `首页 iframe 地址` | 空 | 首页可嵌入的外部页面地址 |
| `隐私政策` | 空 | 隐私政策 |
| `服务条款` | 空 | 服务条款 |
| `站点公告` | 空 | 站点级横幅公告 |

## 侧边栏开关

控制后台侧边栏各入口的显隐，便于按需裁剪界面：
关闭某项只是隐藏入口，不代表关闭后端能力——真正关闭功能要用对应的功能开关。

## SMTP（邮件）

用于发送注册/改密验证码等邮件：

| 设置键 | 默认 | 说明 |
| --- | --- | --- |
| `SMTP Host` | 空 | SMTP 服务器 |
| `SMTP Port` | `587` | 端口 |
| `SMTP 用户名` | 空 | 用户名 |
| `SMTP 密码` | 空 | 密码 |
| `发件人` | 空 | 发件人地址 |

启用邮箱验证前，请先配置好 SMTP，否则验证码无法送达。
