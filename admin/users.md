# 用户管理

在用户管理中，管理员可以查看和调整平台所有用户。

## 能做什么

通过更新用户，可以调整：

- **余额**：手动增减用户余额（如补偿、线下充值）；
- **分组**：调整主分组或分组成员关系，影响计费倍率；
- **管理员标记**：把用户提升为管理员或取消；
- **启用/封禁** 等状态（以界面提供为准）。

## 常见运营操作

**给用户线下充值后加余额**
在用户编辑里增加余额。

**给某用户长期优惠**
把他加入一个低倍率分组（可设到期时间实现限时优惠）。

**提升管理员**
设置管理员。也可通过 `BOOTSTRAP_ADMIN_EMAILS` 在登录时自动提升，见 [初始化设置](/admin/initial-setup)。

**处理疑似滥用账户**
禁用其 API 密钥、调整分组倍率，或按界面提供的封禁能力处理。

::: warning 删除用户
删除用户会影响其关联数据（密钥、日志、订单等）。删除前确认无未结算事项，必要时先导出其日志留档。此操作不可逆。
:::
